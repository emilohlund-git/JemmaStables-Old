Jemma Stables Integritets Policy

Den här Integritetspolicyn beskriver hur din personliga data lagras, används, och delas när du besöker https://www.jemmastables.com/.

Personlig information som vi lagrar

När du besöker våran hemsida så lagrar vi viss information om din enhet per automatik, vilket inkluderar information om din webbläsare, IP address, tidszon och ett antal cookies som är installerade på din enhet, läs våran cookie policy för mer information om det.

När du navigerar våran hemsida så hämtar vi information om de individuella sidor som du besöker, vilka hemsidor eller sökord som referrerat dig till våran hemsida och information om hur du interagerat med våran hemsida. Vi referrerar till denna automatiskt insamlade informationen som 'Enhets Information'.

Vi samlar Enhets Information genom följande tekniker:

- "Cookies" är data filer som sparas på din enhet eller dator och inkluderar ofta en anonym unik identifierare. För mer information om cookies, och hur du kan inaktivera dem, så kan du läsa mer på https://www.allaboutcookies.org/.

- När du besöker våran tidsbokning och bokar en tid, så samlar vi viss information om dig. Informationen som vi samlar är de uppgifterna som du fyller i formuläret när du genomför tidsbokningen. Alltså din epost adress samt ditt namn. Detta för att kunna följa upp vilken tid du har bokat på vilken dag, samt för att kunna skicka bekräftelsemail till dig och visa andra kunder som navigerar till våran tidsbokning att den tiden inte finns tillgänglig längre.

Den här informationen kallar vi för 'Bokningsinformation'. När vi pratar om 'Personlig Information' i den här Intigritetspolicyn, så pratar vi både om Enhetsinformation och Bokningsinformation.

Hur använder vi din personliga information?

Vi använder bokningsinformationen för att generellt kunna uppfölja de bokningar som blir gjorda via hemsida (Inkluderande behandling utav din epost address samt ditt namn).

Vi använder även bokningsinformationen för att kommunicera med dig.

Vi använder enhetsinformationen som vi samlar för att hjälpa skydda oss mot potentiella risker (mer konkret, din IP address), och mer generellt för att förbättra och optimisera våran hemsida (Till exempel genom att analysera hur våra kunder navigerar och interagerar med våran hemsida).

Delning utav din personliga information

Den personliga information som vi delar är din enhets information, vi använder oss utav Google Analytics för att hjälpa oss förstå hur våra besökare använder våran hemsida. Du kan läsa mer om hur Google använder din personliga information här https://www.google.com/intl/en/policies/privacy/. Du kan också tacka nej till att detta sker här https://tools.google.com/dlpage/gaoptout.

Sist, vi får också dela din personliga information för att följa gällande lagar och föreskrifter, för att svara till en stämning, husrannsakningsorder och annan laglig begäran för informationen som vi får till oss, eller för att annars skydda våra rättigheter.

Du kan tacka nej till några utav dessa tjänster genom att besöka 'Digital Advertising Alliance’s' opt-out portal på http://optout.aboutads.info/.

Spåra inte

Notera att vi inte på något sätt justerar våran hemsidas data insamling eller använder dessa metoder när vi ser en 'spåra inte' signal från din webbläsare.

Om du är en Europeisk medborgare så har du rätt att få tillgång din den personliga information som vi lagrar om dig, du kan be om att din personliga information blir rättad, uppdaterad eller borttagen. Om du vill genomföra något utav det här så ber vi dig vänligen att kontakta oss genom kontaktinformationen nedanför.

Uppföljningsvis, om du är en Europeisk medborgare så vill vi poängtera att vi bearbetar din information för att kunna uppfölja de kontrakt som vi kan komma att ha med dig (till exempel om du bokar en tid till våran ridbana). Lagring utav data När du bokar en tid genom våran hemsida så kommer vi behålla din bokningsinformation i våran databas tills du ber oss ta bort den här informationen.

Ändringar

Vi kan komma att behöva uppdatera den här Intigritetspolicyn då och då för att behöva spegla förändringar i hur vi behandlar din information, eller för andra operativa, legala eller reglerande anledningar.

Kontakta oss

För mer information om hur vi behandlar privat information. Om du har frågor eller vill lämna ett klagomål, ber vi dig kontakta oss per mail på info@jemmastables.com.
